🤖 AI Mode Chatbot (LangChain + Mistral + Streamlit)

An interactive chatbot built using LangChain, Mistral AI, and Streamlit that allows users to switch between different AI personalities:

😂 Funny Mode
😢 Sad Mode
😡 Angry Mode

The chatbot maintains conversation history and allows resetting chat to switch personalities seamlessly.

🚀 Features
Multiple AI personalities (Funny / Sad / Angry)
Real-time chat interface using Streamlit
Conversation memory using LangChain messages
Reset chat functionality to switch modes cleanly
Simple and minimal UI

🛠️ Tech Stack
Python
LangChain
Mistral AI
Streamlit
dotenv
